# 📊 Super Store Sales Analysis Dashboard

## 🔍 Project Overview

This project presents an interactive Power BI dashboard built using the Super Store dataset.
The dashboard helps analyze sales performance, profit, and customer purchasing behavior.

---

## 📁 Files Included

* `SuperStoreDashboard.pbix` – Power BI dashboard file
* `dashboard.png` – Dashboard preview image
* `dataset.csv` – Dataset used for analysis
* `superstoresqlquries 
----
## 📌 Key Features

* ✔️ Total Sales, Profit, and Orders KPIs
* ✔️ Profit Margin Analysis
* ✔️ Top & Least Selling Products
* ✔️ Most Profitable Product
* ✔️ Sales by Country & State
* ✔️ Orders by Day and Sub-Category
* ✔️ Interactive Filters (Month, Category, Year)

---

## 📊 Insights

* 📦 Staples is the top selling product
* 🌍 United States generates the highest sales
* 📉 Sunday has the lowest number of orders
